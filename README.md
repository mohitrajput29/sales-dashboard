# Sales Dashboard Redux Project

A modern Sales Analytics Dashboard built using React, Redux Toolkit, Recharts, Context API, and Vite.
# Features

- Sales analytics dashboard
- Revenue charts using Recharts
- Redux Toolkit state management
- Async API handling using createAsyncThunk
- Search and category filtering
- Search debouncing using custom hook
- Dark / Light theme switching
- Theme persistence using localStorage
- Responsive dashboard UI
- Loading state handling
